## Abstract
In controlled environments such as factories, shipyard, airports use many vehicles for various tasks. All the vehicles operating in such environment are non-autonomous and require constant human interaction, so there is a need for autonomous vehicles. Autonomous navigation technologies have the potential to improve efficiency, safety and coordination of vehicles operating in controlled environments. However completely changing all the vehicles to autonomous vehicles is costly and not feasible. So, our project presents a proofof-concept software framework for AI driven path planning designed for controlled environments. The proposed framework focuses on simulation-based proof-of-concept in graph-based environment with modular system design, while A* algorithm is used for calculating optimal and collision-free paths between defined start and goal positions. The system is implemented and evaluated using simulation experiments and a visualization dashboard that visualizes the path planning behaviour along with dynamic replanning. This simulation results demonstrate that classical path planning techniques can effectively support autonomous navigation in controlled environments with low computational overhead. The proposed framework establishes a scalable foundation for future hardwarebased autonomous vehicle systems.
